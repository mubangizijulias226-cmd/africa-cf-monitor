# JoJo Reading Adventure

A simple, interactive single-page reading helper designed for a Primary 2 learner.

## Features
- Random beginner-friendly words.
- Text-to-speech for words and letters.
- Typing check with encouraging feedback.
- Score and streak to keep learning fun.
- Mini story sentence using the current word.
- Tap-to-hear alphabet buttons.

## Run
Open `index.html` in a browser.
